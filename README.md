# CliveBell-THE_AESTHETIC_HYPOTHESIS-
Eine Sammlung von KI generieten Skripten über Emotionen und Futurismus

README: Emotionen & Futurismus Processing Scripts
Übersicht

Dieses Repository enthält eine Sammlung interaktiver Processing-Skripte, die zwei zentrale Themen explorieren:

Emotionen visualisieren – abstrakte, expressive Darstellungen von Stimmungen und Gefühlen.

Futuristische Systeme – dynamische, selbstorganisierte Simulationen inspiriert von Cyberpunk, Sci-Fi und post-impressionistischer Ästhetik.

Alle Skripte kombinieren ästhetische Visualisierung, interaktive Steuerung und logisch geschlossene Systeme, um visuell faszinierende und experimentelle Ergebnisse zu erzeugen.

Skripte
1. Emotionale Visualisierung

Pulsierende Kreise: Darstellung von Stimmungen über Größe, Farbe und Bewegung der Kreise.

Herz / Spiegel der Stimmung: Interaktive Herzform, die Farb- und Bewegungsmuster basierend auf simulierten Emotionen anpasst.

Steuerung: Maus bewegt Elemente, Tastatur verändert Intensität und Puls.

2. Futuristische Simulationen

Futuristisches Netz: Dynamisches, leuchtendes Netz, das Bewegungen von Knotenpunkten und Verbindungen simuliert.

Cyberpunk-Stadt: Neonfarbene, geometrische Architektur mit pulsierenden Lichtern und futuristischem Flair.

3. Selbstorganisierte Systeme

Game of Life (Zellulärer Automat)

Conway’s Game of Life mit interaktiver Zellsteuerung.

Farben pulsieren je nach Generation, Space-Taste pausiert die Simulation.

Boids / Schwarmverhalten

Schwarmintelligenz: Separation, Alignment, Cohesion.

Maus zieht oder stößt Boids ab; Q/W & A/S passen Geschwindigkeit und Wendekraft an.

Diffusions-Limited Aggregation (DLA)

Fraktale Strukturen durch zufällige Partikelbewegung und Aggregation.

Farbmodulation nach Distanz zum Zentrum; W/S passen Partikelgeschwindigkeit an.

Predator-Prey / Lotka-Volterra

Simulation von Räuber- und Beutepopulationen.

Dynamische Populationszyklen sichtbar; C-Taste setzt die Simulation zurück.

Reaction-Diffusion / Turing Patterns

Gray-Scott Modell erzeugt biologisch inspirierte Muster (Streifen, Punkte, Wirbel).

Maus lokal verändert die Substanzkonzentration.

Lindenmayer-System (L-System)

Rekursive Baum-/Pflanzenstrukturen.

Zufällige Winkel, Längen und Farben erzeugen lebendige, natürliche Formen.

Optionen für Strichstärke, Iterationen und dynamische Variation.

Installation & Nutzung

Processing IDE installieren: https://processing.org/download

Repository klonen oder Skripte herunterladen.

Skript öffnen: .pde-Datei in Processing öffnen.

Starten: Play-Button in Processing drücken.

Interaktivität nutzen:

Mausbewegungen, Drag & Drop, Klicks (abhängig vom Skript)

Tastatursteuerung für Parameteranpassungen (siehe Kommentare im Code)

Steuerung (Beispiele)
Skript	Steuerung
Game of Life	Space = Start/Stop, Klick = Zelle umschalten, C = Reset
Boids	Maus = anziehen/abstoßen, Q/W = maxSpeed, A/S = maxForce, Space = Modus
DLA	W/S = Geschwindigkeit, C = Reset
Predator-Prey	C = Reset
Reaction-Diffusion	Maus drag = lokale B-Erhöhung
L-System	Iterationen, Winkel, Strichstärke, Farben variabel im Code
